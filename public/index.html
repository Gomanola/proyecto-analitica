Modelos Analíticos Incluidos

Random Forest (COVID-19)

Predice la supervivencia del paciente en base a su diagnóstico, intervención clínica realizada, cantidad y tarifa facturada. Incluye un módulo de análisis textual clínico (Etapa 1: Detective) y un clasificador entrenado sobre más de 117 mil registros (Etapa 2: Modelo).

Regresión Lineal 1 (Inversiones vs Ventas)

Analiza el comportamiento histórico de ventas y predice de forma automática las ventas del mes siguiente de acuerdo a una progresión uniforme de inversión.

Regresión Lineal 2 - Ejercicios 1 y 2

Modela la relación lineal para estimar el tiempo de entrega de repartidores en base a la distancia, y el tamaño de almacenamiento necesario en base al número de usuarios del sistema.

Estado del Sistema

  • Servidor Backend: Activo (Node.js)
  • Base de Datos Central: SQLite Conectado
  • Modelo RF Precargado: JSON Cargado (50 Árboles)
  • Precisión del Modelo: 70.27% (Cross-Val)

Módulo Detective (NLP Masivo)

El script original de Detective escanea los expedientes clínicos de todos los pacientes en la base de datos para etiquetar la mortalidad de forma masiva antes de entrenar el clasificador.
